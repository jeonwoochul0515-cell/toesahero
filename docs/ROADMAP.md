## 전제 — 우리의 구조적 우위 (다시 정리)

리서치 3건이 일관되게 가리키는 결론:

1. **한국에는 일본의 労働組合 우회 모델이 작동하지 않는다** → 변호사 운영이 사실상 합법적인 유일한 "교섭형 퇴사대행"
2. **2022년 대법원 판결로 노무사 분쟁 대응에 명확한 한계** (고소·민사·형사는 변호사 전속) — 시장 경쟁자(바이바이 등) 대비 명백한 차별화
3. **시장 인지도·이용률은 일본 대비 5~7년 뒤** — 일본 모무리·미야비 모델을 한국 변호사 운영 형태로 적용 가능
4. **2025년 변협 광고규정 강화** — 무료/할인/환불 광고 금지 → 직접 광고 대신 콘텐츠 마케팅·SEO·콘텐츠 PR 중심으로 우회

이 4가지가 1·2·3년차 모든 의사결정의 컴파스가 됩니다.

## 🟢 1년차 (2026.05 ~ 2027.04) — 시장 진입 + 데이터 축적

### Q1 (5월~7월) — MVP 완성 + 초기 트래픽

**제품**:
- ✅ 사이트 + AI 챗봇 베이직 자동화 (Phase 1) 완료 — 통보문 초안 자동 생성
- ⏳ **AI 챗봇 변협 사전 자문 받기** ← (i-boss 등 비공식 해석에서는 사전등록 의무로 보지만, 변협 1차 자료에서는 명시적 등록 절차 미확립 상태) 변협 광고심사위 또는 부산지방변호사회에 서면 자문 받아 운영 안전 확보 권장
- ⏳ **변호사 광고 심의 통과** — 사이트 카피 변협 자체 심의 1회
- ⏳ **개인정보처리방침 / 이용약관 변호사 본인 검토** — 현재 1차 초안 기준

**마케팅 (즉시 시작)**:
- 검색 SEO + 콘텐츠 마케팅
  - "퇴사대행 변호사", "퇴직금 미지급 절차", "사직서 거부", "권고사직 협상" 등 키워드
  - 변호사 본인 명의 정보성 칼럼 (월 4편)
- 칼럼 기고
  - 한경 매거진, 매일경제, 잡코리아 매거진, 부산일보, 국제신문
  - 법률신문, 로앤비 (변호사 인지도)

**KPI**:
- 월 상담 신청 30건 이상
- 베이직 패키지 월 5건 이상
- AI 챗봇 통보문 초안 생성률: 신청 대비 50%+

### Q2 (8월~10월) — 결제 인프라 + 발송 자동화 (Phase 2)

**제품**:
- 💳 **PG 연동** (토스페이먼츠 또는 포트원) — 베이직 199K 즉시 결제
  - 변협 규정상 결제 자체는 가능. 다만 "할인", "환불 보장" 같은 광고 표현은 금지
- 📤 **발송 자동화 (Phase 2)**:
  - 이메일 발송 (Resend / SendGrid) — 회사 측 인사담당자에게 변호사 명의 통보문 PDF 첨부
  - 카카오 알림톡 (Sender ID 인증 필요) — 의뢰인에게 진행 상황 알림
  - 우편 발송 옵션 (등기) — 분쟁 대비
- 📑 **PDF 자동 생성** — 변호사 직인·도장 이미지 포함

**워크플로**:
```
의뢰인 채팅 → AI 통보문 초안 → 의뢰인 위임 신청 → 결제(199K)
  → 변호사 어드민 검토 → 승인 → 자동 발송(이메일+카톡 알림톡)
  → 의뢰인 알림 → 회사 응대 단계
```

**KPI**:
- 결제 완료율: 통보문 검토 요청 대비 30%+
- 발송 자동화율: 변호사 승인 후 5분 내 이메일·카톡 발송 100%
- 베이직 처리 평균 시간: 24시간 이내 (의뢰부터 통보 발송까지)

### Q3 (11월~1월) — 표준 절차 자동화 + 데이터 PR

**제품**:
- 📊 **임금·연차수당 산정 자동 계산기** (표준 패키지 390K)
  - 입력: 월급, 근속기간, 평균 야근시간, 미사용 연차일수
  - 출력: 청구 가능 항목별 금액 추정 (참고용 명시)
  - "예상 청구액"이 아닌 "검토 가능 항목 합산"으로 표현 (변협 규정)
- ✉️ **내용증명 자동 생성** (표준 패키지) — 임금 청구 내용증명
- 🔔 **표준 패키지 워크플로** — 베이직과 같은 검토/승인 자동화

**마케팅 (모무리 벤치마크)**:
- 📢 **데이터 PR**: 분기별 "퇴사 이슈 통계" 공개
  - "2026년 Q3 의뢰 통계 — 직장 내 괴롭힘 35%, 임금 체불 22%, 권고사직 17%"
  - 모무리가 일본에서 1.5만 명 데이터 공개로 신뢰 확보한 모델 그대로
- 🎬 **유튜브 채널 본격화**
  - "법률사무소 청송 김창희 변호사" 채널
  - 5~10분 정보성 영상 (사직서 작성법, 퇴직금 산정, 직장갑질 신고 절차)
  - 광고 X — 정보 전달 형식 → 변협 안전권

**KPI**:
- 월 상담 신청 100건 이상
- 표준 패키지 월 20건 이상
- 콘텐츠 마케팅 SEO 노출: 주요 키워드 검색 1페이지 진입

### Q4 (2월~4월) — 분쟁 대응 사례 누적 + 1년 회고

**제품**:
- ⚖️ **분쟁 대응(990K) 패키지 사례집** 누적

- 🔍 **사건 추적 대시보드** (어드민) — 진행 중 분쟁 케이스 칸반 뷰

**B2B 진입 (모무리 벤치마크)**:
- 🤝 **노무법인 협업 협약** — 노무사가 처리 못하는 분쟁 사건을 본 사무소로 의뢰

## 핵심 결론 (1줄)

**"한국 시장의 5~7년 시간차 + 일본 노조 우회 모델 부재 + 2022 대법 판결로 노무사 한계가 명확한 지금이, 변호사 운영 퇴사대행 브랜드를 First-mover로 만들 수 있는 유일한 시점."**

— 1년차에 검색·콘텐츠로 인지도, 2년차에 B2B 본업화, 3년차에 플랫폼·네트워크.
